# Баянбараатын Дөмөн Аялгуу Танилцуулгын Вэбсайт

This is a replica of the Orkhon Valley Cultural Landscape World Heritage Site website (https://orkhonvalley.gov.mn/).

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Interactive Carousel**: Auto-rotating hero carousel with 18 slides
- **Navigation Menu**: Multi-level dropdown navigation with mobile hamburger menu
- **Content Sections**: 
  - Heritage (ДӨ-Орхоны хөндий)
  - Administration (Хамгаалалтын захиргаа)
  - Tourism (Аялал жуулчлал)
  - News (Онцлох мэдээ)
- **Modern UI**: Porto theme-inspired styling with smooth animations
- **Smooth Scrolling**: Smooth scroll navigation between sections
- **Touch Support**: Swipe gestures for carousel on mobile devices

## How to Use

1. Open `index.html` in a web browser
2. The website will work offline, but requires internet connection for Font Awesome icons
3. All functionality is included:
   - Carousel auto-rotates every 4 seconds
   - Navigation menu works with dropdowns
   - Mobile menu accessible via hamburger icon
   - Smooth scrolling between sections

## Customization

You can customize the website by:
- Modifying colors in `styles.css` (CSS variables in `:root`)
- Adding more content sections in `index.html`
- Adjusting carousel timing in `script.js` (currently 4000ms)
- Adding images to the carousel slides
- Extending the news articles

## Notes

- The website uses Font Awesome icons from CDN
- All text is in Mongolian (Cyrillic script)
- The design follows the Porto theme style
- The carousel can be paused by hovering over it
