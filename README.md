# webduuri-maker

Товч танилцуулга
-----------------
webduuri-maker нь frontend/вэб төсөл бөгөөд JavaScript, HTML, CSS ашиглан бүтээгдсэн. Төслийн гол зорилго нь хурдан, хөнгөн веб интерфэйс ба энгийн клиент талын логикийг агуулсан жишээ аппликейшн байх шаардлагатай үед ашиглахад тохиромжтой.

Хэлний бүрэлдэхүүн (өлсөл)
-------------------------
- JavaScript: 36.5%
- HTML: 32.3%
- CSS: 31.2%

Шаардлагатай зүйлс
------------------
- Орчин: Дээрх файлууд нь статик вэб проект тул хэн дуртай вэб хөтчөөр нээж үзэж болно.
- (Сонголт) Node.js хэрэв dev server ашиглах бол: https://nodejs.org/

Бүтээн байгуулалт, ажиллуулах
-----------------------------
Энгийнээр:
1. Репозиторыг клаон хийх:
   - git clone https://github.com/webduuri-maker/-  # репозитортой холбоосыг өөрчлөх шаардлагатай байж болно
2. index.html файлыг веб хөтчид нээнэ үү.

Хэрэв развопмент/хөгжүүлэлтийн орчин ашиглах бол (жишээ):
- Хэрэв пакет.json, dev server байгаа бол:
  - npm install
  - npm start

Мөн тохиромжтой хурдан dev server:
- npx serve .   (эсвэл)
- npx http-server .

Файлын бүтэц (жишээ)
---------------------
- index.html      — Гол хуудас
- src/            — JavaScript ба CSS эх код
- assets/         — Зураг, фонтууд гэх мэт
